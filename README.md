LLM-based chatbot (GPT-4o) using Retrieval-Augmented Generation to answer student queries, with a retrieval pipeline across PDF, web, and SQL sources and a Streamlit interface.

**🔬 [Intelligent Microscopy Assistant](https://github.com/hamdimoh/intelligent-microscopy-analysis-assistant)**
Web-based computer vision app for bacteria detection in microscopic images, with a React frontend, chatbot integration, and PDF/XML export.

Web application for automated event scheduling built with C# and ASP.NET Core MVC.
